Patient Management System(Spring Boot)
A simple and clean patient records management system built with Spring Boot.

Features
- Add new patient
- View all patients
- Update the patient attributes
- Delete a patient record

Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- Maven
- MySQL
